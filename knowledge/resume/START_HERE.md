# OneMind Resume Pack

This directory is the AI entry point for resuming the OneMind project.

Always read the following files in order.

1. PROJECT_STATUS.md
   - Current project status
   - Current milestone
   - Current version

2. ONEMIND_CONTEXT.md
   - Project overview
   - Architecture philosophy
   - Design principles
   - Repository structure

3. CURRENT_HANDOFF.md
   - Latest milestone handoff
   - Decisions made
   - Remaining work
   - Next tasks

After reading all files above,

Resume the project from the current milestone.

Do not redesign completed milestones unless explicitly requested.

Architecture documents remain the Source of Truth.
