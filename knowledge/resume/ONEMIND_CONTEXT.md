# OneMind Context

## Project Identity

Project Name:
OneMind

Vision:

Many Agents. One Mind.

Mission:

Create an AI Operating Platform where multiple agents collaborate through shared knowledge, memory, context and orchestration.

---

## Architecture Philosophy

OneMind is designed as an AI Operating Platform.

The platform is built around the idea that many specialized agents can work together as one coordinated intelligence.

Architecture drives technology decisions.

Technology must not drive architecture decisions.

---

## Core Principles

1. Platform First

Build a platform, not a collection of tools.

2. Multi-Agent by Design

Agents are first-class citizens.

3. Shared Knowledge

All agents can access common knowledge.

4. Shared Memory

Memory is a platform capability.

5. Shared Context

Context is a platform capability.

6. Human-in-the-Loop

Humans remain part of critical workflows.

7. Documentation as Code

Architecture and documentation are versioned assets.
