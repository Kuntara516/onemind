# Technology Architecture

> OneMind Enterprise Architecture

---

## Purpose

Technology Architecture กำหนดมาตรฐานและแนวทางการเลือกเทคโนโลยีที่ใช้สร้าง OneMind Platform

เอกสารในโฟลเดอร์นี้อธิบาย Technology Capabilities, Technology Standards และ Reference Technology Stack โดยไม่ผูกกับผู้ผลิตหรือผู้ให้บริการรายใดรายหนึ่ง

Technology Architecture ทำหน้าที่เชื่อมระหว่าง Logical Architecture และ Deployment Architecture
