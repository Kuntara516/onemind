# Architecture Governance

## Purpose

กำหนดมาตรฐาน กฎเกณฑ์ และแนวทางการกำกับดูแลสถาปัตยกรรมของ OneMind เพื่อให้การพัฒนาระบบเป็นไปในทิศทางเดียวกัน มีคุณภาพ สามารถตรวจสอบได้ และรองรับการเติบโตในระดับ Enterprise

Governance เป็นกลไกที่เชื่อมโยงระหว่าง Business, Architecture, Security และ Software Engineering
