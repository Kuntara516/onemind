# Preamble

OneMind exists to transform organizations from fragmented systems into unified intelligence.

We believe that software should do more than automate tasks.
Software should preserve knowledge, coordinate people, empower intelligent agents, and continuously improve how organizations operate.

OneMind is not merely an application.

It is an AI Operating Platform designed to unify people, knowledge, workflows, assets, and intelligent agents into a single operational mind.

Every decision, every document, every workflow, every line of code, and every AI agent shall exist to serve this purpose.

This document is the highest governing document of the OneMind Project.

All engineering decisions, architectural designs, specifications, implementations, and governance documents shall comply with the principles defined herein.

# Article V
## Rule Zero
### Single Source of Truth (SSOT)

Every important fact shall have exactly one authoritative source.

Documentation is the source of truth.

Reference documents instead of duplicating information.

Update the source first.

Everything else follows.
